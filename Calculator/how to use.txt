Users must use a compatible C Compiler to run the program. </br>Upon running the program, users will be shown the main menu consisting of basic and scientific calculators. </br>An additional option to know about the developers will also be shown. </br>In Basic Mode users can perform regular arithmetic operations like addition, subtraction, multiplication, division etc. </br>During Addition & Subtraction users must enter a digit followed by the relevant character (+, -) and after the last input user must input an equal sign (=) to finish the operation. </br>To calculate percentage users need to enter a value and the percentage they want to calculate from the value. </br>In Scientific Mood users can perform some additional mathematical operations. </br>In every operation the result will be shown with 6 decimal places after the decimal point if needed. </br>To solve equations users must input only the coefficient of the relevant terms. </br>Under the section “Advance Calculations” users can count factorials only up to 19. </br>Permutations and Combination counting is also limited to a certain degree. </br>During the temperature conversion the result will be shown only up to two decimal places. </br>After performing any kind of operation users will be asked if they want to perform the calculation again. </br>Upon entering “Y/y:”, users can perform the same operation again and by entering “N/n” users can go back to the last menu they accessed. </br>By entering “0” at any menu or sub-menu users can return to the previous menu. </br>If the users close the terminal the program will be terminated. </br>All the records of the calculations will be stored in a text file under the name “calculation_history”. </br>Users can open this file to checkout the results of their previous calculations.
